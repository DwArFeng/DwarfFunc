# DwarfUtil for Java - DwArFeng的Java功能代码

## DwarfUtil 是干什么的？

DwarfUtil (DwArFeng's java func) 是DwArFeng在编写Java程序时总结的一些工具方法和一些常用的类。

## DwarfUtil包含哪些内容？

DwarfUtil 包含一些基本的，用于各种程序中的快捷的或者具有功能性的工具包，也含有用于数学、物理学等其它方面的功能
