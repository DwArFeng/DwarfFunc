# DwarfFunction for Java - DwArFeng的Java功能代码

## DwarfFunction 是干什么的？

DwarfFunction (DwArFeng's java func) 是DwArFeng在编写Java程序时总结的一些工具方法和一些常用的类。

## DwarfFunction包含哪些内容？

>dfunc 整个DwarfFunction包
>>cls 与Class有关的方法或类。
>>cna 与集合或数组有关的方法或类。
>>comp 常用的比较器。
>>dt 调试工具。
>>gui 与界面有关的类。
>>infs 常用接口定义。
>>io 输入/输出有关。
>>num 与数字方法有关。
>>prog 与程序框架有关。
>>> mvc 与MVC程序构架有关。
>>str 与字符串有关。
>>threads 与线程有关。
